ตั้งค่า API
เชื่อมต่อกับ Google Sheets ผ่าน Apps Script
1

สร้าง Google Sheet ใหม่

ไปที่ sheets.google.com → สร้าง Spreadsheet ใหม่ → ตั้งชื่อตามใจ เช่น "ร้านขนม DB"

2

วาง Apps Script

ใน Sheet → เมนู Extensions → Apps Script → ลบโค้ดเดิมออก → วางโค้ดจากไฟล์ Code.gs ที่ให้มา → กด Save

3

Deploy เป็น Web App

ใน Apps Script → Deploy → New deployment
· Type: Web app
· Execute as: Me
· Who has access: Anyone
→ กด Deploy → Copy URL ที่ได้

⚠ ครั้งแรก Google จะขอ Permission → กด "Review permissions" → เลือก account → "Allow"
4

ใส่ URL ด้านล่าง

วาง Web App URL ที่ได้จาก Deploy ตรงนี้ — ระบบจะบันทึกไว้ใน browser และใช้เชื่อมต่อทุกครั้ง
